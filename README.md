# MovieApp

This is a simple Android application that allows you to search for movies and save them to a local database.

## Features

*   Search for movies using the search bar.
*   View a list of movies with their posters and titles.
*   Save your favorite movies to a local database.
*   View your saved movies on a separate screen.

## How to Use

1.  Open the application.
2.  Use the search bar to find movies.
3.  Click the "Search" button to see the results.
4.  Click the "Save" button to save a movie to your local database.
5.  Navigate to the "Saved Movies" screen to see your saved movies.
